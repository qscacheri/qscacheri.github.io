Portfolio

Groove Pizza Plug-In

The Groove Pizza Plug-In is an advanced audio plug-in version of the NYU MusEDLab’s web-app of the same name that has been used by hundreds of thousands of people. The Groove Pizza is a piece of educational software that allows users to learn the fundamentals of rhythm. The plug-in version allows the user to easily make beats in their recording software and by necessity was completely rebuilt from scratch.

Magenta Beats

Magenta Beats is an audio plug-in that uses Google’s Magenta machine learning library to create new drum beats based on the user’s original inputted beat. The machine-generated pattern can then be fed back into itself to create more unique beats.

aQWERTYon Plug-In

The NYU MusEDLab’s aQWERTYon web app allows users to jam along with their favorite YouTube videos. The app teaches both improvisation and how to determine the key of a song. The aQWERTYon Plug-In is a more advanced version rebuilt from the ground up that allows users to play from within their recording software.

Sound Garden

The Sound Garden is an online, virtual reality, collaborative audio experience and can be found here.

RO-MC-001 (Remote-Operated Mic Clip)

The Remote-Operated Mic Clip allows a user to change the angle of a microphone from anywhere using a Bluetooth remote. The clip was specifically designed for concert recordists with tall mic stands or boom operators with long poles.

Procedural Piet

Procedural Piet generates works of art in the style of painter Piet Mondrian. Click the image below to generate a unique piece of art.

Word Tree

The Word Tree program uses a large string of text to create a unique tree starting with the first word of the text as the trunk of the tree. The remaining branches are created using the subsequent words or series of words. The user can control the color combinations of each tree.

Black Box

Black Box is a web based game inspired by the iOS app of the same name, that requires players to think outside the box in order to solve challenging puzzles.
